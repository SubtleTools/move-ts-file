## It even updates barrel row indexes!

When you move files that are re-exported by barrel files, move-ts-file intelligently updates the entire dependency tree to maintain clean import paths.

```bash
# Move a utility file that's re-exported by a barrel
npx move-ts-file src/utils/validation.ts src/shared/validation.ts
```
