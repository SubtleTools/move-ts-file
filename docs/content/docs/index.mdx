## It even updates barrel exports!

When you move files that are re-exported by barrel files, move-ts-file intelligently updates the entire dependency tree to maintain clean import paths.

<Steps>

## What Happens

1. **File moved**: `src/utils/validation.ts` â†’ `src/shared/validation.ts`
2. **Export removed from utils**: The export is removed from `src/utils/index.ts`
3. **New barrel created**: A new barrel file `src/shared/index.ts` is created
4. **Imports updated**: Consumer files are updated to import from the new location
5. **Import styles preserved**: Each file maintains its original import style
