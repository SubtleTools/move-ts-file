## Installation

You can install `move-ts-file` globally with your preferred package manager or use it directly without installation.

import { Tabs, TabItem } from '@astrojs/starlight/components';

### Global Installation

<Tabs>
<TabItem label="npm">
```bash
npm install -g move-ts-file
```
</TabItem>
<TabItem label="yarn">
```bash
yarn global add move-ts-file
```
</TabItem>
<TabItem label="pnpm">
```bash
pnpm add -g move-ts-file
```
</TabItem>
<TabItem label="bun">
```bash
bun add -g move-ts-file
```
</TabItem>
</Tabs>

Then use it anywhere:

```bash
move-ts-file src/old-file.ts src/new-file.ts
```

### Use without Installation (Recommended)

No installation required - use it directly with your package manager's runner:

<Tabs>
<TabItem label="npx">
```bash
npx move-ts-file src/old-file.ts src/new-file.ts
```
</TabItem>
<TabItem label="yarn dlx">
```bash
yarn dlx move-ts-file src/old-file.ts src/new-file.ts
```
</TabItem>
<TabItem label="pnpm dlx">
```bash
pnpm dlx move-ts-file src/old-file.ts src/new-file.ts
```
</TabItem>
<TabItem label="bunx">
```bash
bunx move-ts-file src/old-file.ts src/new-file.ts
```
</TabItem>
</Tabs>

## Requirements

- **Node.js 18+** or **Bun 1.0+**
- **TypeScript project** with `.ts` or `.tsx` files
- Valid TypeScript syntax (the tool uses TypeScript AST parsing)

## Verify Installation

Check that the tool is installed correctly:

```bash
move-ts-file --version
```

You should see the current version number.
